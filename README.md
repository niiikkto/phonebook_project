## 📝 Описание проекта

**Телефонная книга** - это консольное приложение на Python, которое позволяет пользователям эффективно управлять своими контактами. Приложение предоставляет полный набор CRUD операций (Create, Read, Update, Delete), а также дополнительные функции, такие как поиск по частичному совпадению номера телефона и экспорт/импорт данных в CSV формат.

### Основные возможности:
- ✨ Интуитивно понятный консольный интерфейс
- 💾 Автоматическое сохранение данных в JSON файл
- 🔍 Поиск контактов по имени или номеру телефона
- 📤 Экспорт контактов в CSV для использования в других программах
- 📥 Импорт контактов из CSV файлов
- 🏷️ Поддержка дополнительной информации (email, адрес)

## 💻 Установка

### Требования
- Python 3.7 или выше
- Git (опционально, для клонирования репозитория)

### Способ 1: Клонирование репозитория

```bash
# Клонируйте репозиторий
git clone https://github.com/ВАШ_username/phonebook_project.git

# Перейдите в директорию проекта
cd phonebook_project

# Запустите приложение
python main.py
```


## 🐳 Docker

### Запуск через Docker

#### Предварительные требования

- Установленный Docker Desktop: https://www.docker.com/products/docker-desktop/
- Docker Compose (входит в Docker Desktop)

#### Сборка Docker-образа

```bash
# Сборка образа
docker build -t phonebook .

# Проверка, что образ создан
docker images | grep phonebook
```
