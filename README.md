# Телефонная книга (PhoneBook)

Простое консольное приложение для управления контактами на Python.

## Возможности

- Добавление новых контактов (имя, телефон, email, адрес)
- Поиск контактов по имени
- Просмотр всех контактов
- Обновление информации о контактах
- Удаление контактов
- Автоматическое сохранение данных в JSON файл

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.7+
2. Склонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd phonebook_project
