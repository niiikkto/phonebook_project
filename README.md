# 📞 Телефонная книга (Phone Book)

Консольное приложение на Python для управления контактами с сохранением данных в JSON.

## ✨ Возможности

- ✅ Добавление, редактирование, удаление контактов
- ⭐ Избранные контакты
- 🔍 Поиск по имени, телефону, email
- 💾 Автоматическое сохранение в JSON файл
- 📊 Удобный табличный вывод

## 🚀 Быстрый старт

```bash
# Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/phonebook.git

# Перейдите в папку проекта
cd phonebook

# Запустите приложение
python main.py
